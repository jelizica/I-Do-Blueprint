## üö® Common Mistakes

### ‚ùå DON'T: Use raw shade values

```swift
// ‚ùå WRONG
.foregroundColor(BlushPink.shade600)
.background(BlushPink.shade600.opacity(0.15))
```

### ‚úÖ DO: Use semantic colors

```swift
// ‚úÖ CORRECT
.foregroundColor(SemanticColors.primaryAction)
.background(SemanticColors.backgroundTintBlush)
```

---

### ‚ùå DON'T: Hardcode opacity

```swift
// ‚ùå WRONG
.opacity(0.15)
.opacity(0.1)
.opacity(0.5)
```

### ‚úÖ DO: Use Opacity enum

```swift
// ‚úÖ CORRECT
.opacity(Opacity.light)
.opacity(Opacity.subtle)
.opacity(Opacity.medium)
```

---

### ‚ùå DON'T: Color-only status

```swift
// ‚ùå WRONG (Color blind unsafe)
Text("Confirmed")
    .foregroundColor(SemanticColors.statusSuccess)
```

### ‚úÖ DO: Color + Icon

```swift
// ‚úÖ CORRECT (Color blind safe)
HStack {
    Image(systemName: "checkmark.circle.fill")
    Text("Confirmed")
        .foregroundColor(SemanticColors.statusSuccess)
}
```
